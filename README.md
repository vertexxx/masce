# Markus Autobahn Speed Calculator for EVs

## Calculation
Estimate vehicle energy consumption as a function of driving speed and calculate minimal charging times.
The tool determines the optimal cruising speed to maximize overall average travel speed, taking both driving and charging (including starting the charge) phases into account.
Vehicle presets can be selected from the menu in the top-right corner.
